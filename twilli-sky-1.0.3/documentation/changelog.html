Version 1.0.3 - top

December 2014

Changes/Updates:

  • Update of Bootstrap to v3.3.1
  • Update of Masonry to v3.2.2
  • Minor optimisations and fixes.

Completely Replaced Files:

Replace your version of the following files with the new version of the following files.

  • bootstrap/
  • plugins/masonry/

Updated Files:

First keep a separate backup of your current version of these files. Then it is recommended to use a file comparison tool (ex: WinMerge) to separately merge the updated files with your current version of the same files.

  • css/style-theme.css
    New/Updated CSS rules:
    • Lines 2675 - 2688
    • Lines 2783 - 2787
  • css/style-theme.min.css - if you have made extra changes to style-theme.css, you'll need to re-minify it again using a tool like CSS Minifier
  • js/functions.js
    Updated functions:
    • validate_and_submit_forms
  • js/functions.min.js - if you have made extra changes to functions.js, you'll need to re-minify it again using a tool like Prepros
